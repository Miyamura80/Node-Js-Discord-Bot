**Node JS Discord Bot**

Before you start, you must add a `config.json` file to the root directory, which may look something like the following:


config.json
```json
{
	"prefix": "!",
	"token": "your token here",
	"passwords_array": ["pass1", "pass2", "pass3", "pass4"],
	"secret_passcodes": {
		"sp1": 12414,
		"sp2": 19183
	}
}
```
